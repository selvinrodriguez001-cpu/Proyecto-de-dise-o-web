Como crear un repositorio en GitHub

Aprendamos paso a paso a crear, configurar y subir nuestros proyectos a GitHub

Introduccion

GitHubes una plataforma que permite almacenar, compartir y colaborar en proyecos de desarrollo. Los repositorios son el corazon de GitHub, ya que ahi se guarda el codigo y los archivos de los proyectos.

Pasos para crear un repositorio

  1. Paso1: Ingresa a la pagina principal de GitHub.com e inicia sesion en tu cuenta.
  2. Paso2: Haz clic en el boton "New" o "Nuevo repositorio que aparece en la parte superior izquierda.
  3. Paso 3: Escribe el nombre de tu repositorio, por ejemplo mi-primer-repositorio, y agrega una breve descripcion del proyecto.
  4. Paso 4: Elige si el repositorioseraPublico o Privado
  5. Paso 5: (Opcional) Marca la casilla "Add a README file" para incluir una presentacion inicial del proyecto.
  6. Paso 6: Haz clic en el botn verde"Create repository" para finalizar la creacion.

Como subir archivos desde Visual Studio Code

Una vez creado tu repositorio, podemos centrarlo con tu proyecto local desde Visual Studio Code. Abre la terminal e ingresa los siguientes comandos: 

git init
                git add .
                git commint -m "Primer commit"
                git branch -M main
                git remote add origin
                https://github.com/tuusuario/mi-primer-repositorio.git
                git push -u origin main

Esto enviara todos sus archivos al repositorio remoto en github

Conclusion

Crear u repositorio en GitHub es el primer paso para compartir tu codigo, colaborar con otros desarrolladores y mantener un control profesional de nuestros proyectos. Siguiendo estos pasos, tendremos nuestro repositorio listo y funcionando en minutos.
